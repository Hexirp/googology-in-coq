# googology-in-coq

巨大数論を Coq で形式化します。

## 特徴

googology-in-coq は、いくつかの特徴を持ちます。

* 殆どの定理において、証明の Gallina の項としての表現が明瞭になるように、ライブラリが組み立てられている。
* 表現が同じだが意味付けが異なる定義を転用せず別々に定義する。
* 明示的に universe level を扱う。
* 一貫している命名規則を持つ。
* Homotopy Type Theory の知見を取り入れている。
* 日本語で開発が行われている。
